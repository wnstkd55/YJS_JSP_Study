		객체 생성 어노테이션( @Component )
		의존성 주입 어노테이션 (DI)
			1. @AutoWired : 주로 변수 위에 설정, 해당 타입의 객체를 찾아서 주입 (Spring)
			2. @Qualifier : 특정 객체의 이름을 이용해서 의존성 주입할때 사용.(Spring)
			3. @Resource : @Autowired와 @Qualifier를 합친 기능 (java)
			4. @Inject : @Autowired와 동일한 기능(java)
